# user-input
this will give you user input of any data type you want

-------------------------

for string use str()

-------------------------

for unsigned integer use:

unsign8() for u8

unsign16() for u16

unsign32() for u32

unsign64() for u64

-------------------------

for signed and unsinged integer both use int()

sign8() for i8

sign16() for i16

sign32() for i32

sign64() for i64

-------------------------


for usize use usizeint() 

-------------------------

for character use chara()

-------------------------

for floating point use

float32() for f32

float64() for 64
